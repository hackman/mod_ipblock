A Module for Apache Version 1.3 or later


Apache Module mod_ipblock

Available Languages:  en 

Description: Apache module aimed at preventing or detecting DoS attacks from a single IP.
Status:Extension
Module Identifier:ipblock_module
Source File:mod_ipblock.c
Compatibility:Version 1.3 and later

Summary

mod_ipblock aims at preventing or detecting DoS attacks from a single IP.

It does so by counting the requests from a single IP that are stored in the scoreboard at the moment.

And if the count is higher then the limit set in the configuration it executes a script.


IPBlockLimit Directive

Description:Defines the maximum number simmultaneous connections from a single IP address.
Syntax:IPBlockLimit number
Context:server config
Override:FileInfo
Status:Base
Module:mod_ipblock
Compatibility:Apache 1.3 and later

This directive sets the limit of simmultaneous connections from a single IP address.
When this limit is reached, the module executes the command defined by IPBlockCmd.

Examples

# Maximum number of simmultaneous connections from a single IP address
IPBlockLimit 1000

IPBlockCmd Directive

Description:Defines a command that will be executed when the limit is reached.
Syntax:IPBlockCmd path
Context:server config
Override:FileInfo
Status:Base
Module:mod_ipblock
Compatibility:Apache 1.3 and later

This directive sets the command that will be executed when the limit is reached.

Examples

# Command that will be executed when the limit is reached
IPBlockCmd "/usr/local/bin/ipblock"
